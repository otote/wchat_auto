# wchat_auto
基于itchat和图灵机器人制作的自动回复微信机器人

运行环境：python3
需安装itchat模块。<br>

安装命令：
==
  pip install itchat<br><br><br><br>
可以挂到服务器。<br>
linux关闭终端后后台会被关闭，机器人会停止运行。可以用命令：nohup python what_auto.py 让脚本在后台运行。关闭终端后不会被关闭。但是第一次需要手动运行。
因为要扫码登录。
